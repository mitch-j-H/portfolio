Talabanane

Visit the site

Talabanane is a site that acts as a central location for users to find recreation/social activities from various vendors and by buying credits spend them over a wide range of activities. My brief was to create an admin tool to review users survey responses and connections they made during activities where they participated together. The tool would allow the communication team to have leads to populate activities with like minded people who enjoyed similar activities.

PHP Javascript MariaDB WordPress

Sortir.fr

Sortir.fr was a group project to create a platform for students of the school to create and manage events over various campuses. The brief required creating a database schema and data modelling, a backend to manage the business logic and a front end for the users to interact with. Particular key points were implementing best practices for data and user security, managing a wide range of users with different roles and managing event data(dates, max attendance, locations etc).

Symfony PHP MySQL Javascript

ENI-Encheres

ENI-Encheres was another group project that required building an auction site similar to Ebay. This project came with various challenges particularly writing the whole project without a framework and using JSPs. Highlights of the projects were creating users objects and managing data flow through the application with data validation and security being the primary focus points. Also requiring managing states of items and real time tracking of the clients account balance which was a points based system.
